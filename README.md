# 초성 검색

한국어 초성으로 단어를 검색하는 웹앱. Vite + Vanilla JS.

카테고리별(음식, 동물, 영화, 속담 등 30여 개)로 단어 데이터가 구성되어 있고, `ㄱ`, `ㄷㅇ`, `ㅅㅈ` 같이 초성만 입력하면 해당하는 단어를 전부 찾아준다.

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행 (http://localhost:5173)
npm run dev

# 빌드
npm run build
```

## 기능

- **초성 검색** — 초성 입력 시 짧은 순으로 정렬
- **단어 추가** — 카테고리 헤더 클릭하거나 편집 패널에서 추가 (쉼표로 여러 개 가능)
- **단어 수정** — 편집 패널에서 기존 단어 선택 후 변경
- **주제 추가** — 새 카테고리 직접 생성
- **JSON 내보내기/가져오기** — 데이터 백업 및 복원
- **로컬스토리지 저장** — 변경사항 자동 저장, 앱 기본 데이터와 자동 병합

## 데이터 수정

단어 데이터는 `src/data.js`에서 관리. 카테고리별 문자열 배열 형태.

```js
export const DEFAULT = {
  "음식": ["김치", "된장찌개", ...],
  "동물": ["강아지", "고양이", ...],
  // ...
}
```
